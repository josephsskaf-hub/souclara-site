# souclara — site da Clara

Landing page do **souclara.com.br** (recepcionista de IA por WhatsApp para clínicas).

## Como funciona o deploy
Este repositório está conectado ao Netlify. **Toda alteração no `index.html` que for enviada (push) pro GitHub publica o site sozinho** — sem arrastar nada, sem terminal.

## Arquivos
- `index.html` — a landing completa (página única, sem dependências).
- `netlify.toml` — configuração de publicação.

## Para editar
Edite o `index.html` (no editor do GitHub ou localmente) e faça o commit/push. O Netlify detecta e publica em segundos.
